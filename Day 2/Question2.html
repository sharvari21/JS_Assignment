The JavaScript method toString() converts an array to a string of (comma separated) array values.

2. join():

The join() method also joins all array elements into a string. It behaves just like toString(), but in addition you can specify the separator. Here we will separate the array elements with " ** ".

3. pop():

The pop() method removes the last element from an array. And it returns the value that was popped out.

4. push():

The push() method adds the new element to an array. And it returns the new array length. The new added element will always be added at the end. push() methods takes an argument.
